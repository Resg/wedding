# Sergey & Zhanneta — Next.js приглашение + RSVP

Этот проект собран из идеи/структуры вашего HTML-приглашения (разделы типа *Главная / Приглашение / Опрос / Расписание / Карта*) и переписан на Next.js с собственной формой и записью в БД.  
В оригинальном HTML были подключены bootstrap/jquery и скрипты шаблона, включая блок обратного отсчёта — тут это заменено на современную реализацию без лишнего (отсчёт мы убрали).  

## Быстрый старт (yarn)

```bash
yarn
cp .env.example .env
yarn prisma:generate
yarn prisma:migrate
yarn dev
```

Открой: http://localhost:3000

## Где менять тексты/ссылки/картинку

- `src/lib/config.ts` — имя пары, дата/время, название ресторана, ссылка на Яндекс.Карты
- Картинка первого блока/карточки: `public/images/invite.png` (можно заменить своим файлом)

## RSVP (форма)

- Страница формы: `/rsvp`
- После отправки редирект на: `/thanks/<id>` (там отображаются введённые данные)

## Админка

- `/admin?token=ВАШ_ТОКЕН`
- Токен задаётся в `.env` (переменная `ADMIN_TOKEN`)
